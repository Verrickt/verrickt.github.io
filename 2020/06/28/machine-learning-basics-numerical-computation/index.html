Source of errors

Overflow and Underflow

计算机的内存是有限的，而有限的内存上无法实现无限精度的数值计算，因此数值计算是有误差的。

接近零的非零数值因为舍入变为0称为Underflow。

超出表示范围的wrap-around称为Overflow

  • positive-overflow 指超过范围的正数被截断为负数
  • negative-overflow 指超过范围的负数被截断为正数
